# 📚 BookBot — Groq AI (100% Free)

AI-powered book chatbot using Groq API — completely free!

## Folder Structure

```
book-chatbot-groq/
├── index.html       ← Open this in browser
├── style.css        ← Design (dark/light mode)
├── app.js           ← Main logic
├── api.js           ← Groq API calls
├── fileHandler.js   ← PDF & TXT reading
└── storage.js       ← Save chat history
```

## How to Use

1. Go to https://console.groq.com/keys
2. Sign up FREE — no credit card needed
3. Copy your API key
4. Open index.html in browser
5. Paste API key in the top bar
6. Create a folder → Upload PDF/TXT → Chat!

## Why Groq is Free?

- 14,400 free requests per day
- No credit card required
- Fast responses (faster than ChatGPT!)
- Model used: llama-3.3-70b-versatile
